# SUN PEPE — Landing Page Plugin

WordPress plugin providing a Hebrew RTL landing page for **SUN PEPE**, a premium kosher pizzeria at הרצל 71, רמת גן.

---

## התקנה ראשונית (למתחילים)

### שלב 1 — העלאת הפלאגין לוורדפרס

1. פתחו את לוח הבקרה של וורדפרס (WP Admin)
2. לכו ל**פלאגינים → הוסף חדש → העלה פלאגין**
3. העלו את תיקיית הפרויקט כ-ZIP, או העתיקו את תיקיית `SUNPEPE` ישירות אל `wp-content/plugins/` בשרת שלכם
4. הפעילו את הפלאגין **SUN PEPE Landing Page**

### שלב 2 — יצירת הדף

1. לכו ל**עמודים → הוסף חדש**
2. תנו לדף שם (למשל: "דף הבית")
3. מצד ימין, תחת **תבנית עמוד**, בחרו: **SUN PEPE Landing Page**
4. לחצו **פרסם**
5. הגדירו אותו כ**דף הבית** תחת הגדרות → קריאה → דף בית סטטי (אם רוצים)

### שלב 3 — התקנת ACF (לעריכת מחירים)

> **מה זה ACF?** Advanced Custom Fields — פלאגין חינמי שמאפשר הוספת שדות (כמו מחיר) לפריטי תפריט.
> ללא ACF, כל המחירים יוצגו כ-₪0 — הדף עדיין יעבוד תקין.

1. לכו ל**פלאגינים → הוסף חדש**
2. חפשו: `Advanced Custom Fields`
3. התקינו והפעילו את הגרסה **החינמית** (מאת Elliot Condon / WP Engine)

### עריכת פריטי התפריט ומחירים

> **הערה:** יש להתקין ACF (ראה שלב 3 למעלה) כדי לערוך מחירים.

#### הוספת פריט תפריט חדש

1. לכו ל**SUN PEPE → פריטי תפריט → הוסף פריט**
2. **כותרת** = שם הפריט (למשל: "פיצה מוצרלה")
3. **קטגוריה** = בחרו מהרשימה (פיצות, סלטים, מאפים, פסטות, שתייה, תוספות)
4. **מחיר** = הכניסו את המחיר בשקלים (ברירת מחדל: 0)
5. **תיאור קצר** = אופציונלי — תיאור קצר של הפריט
6. לחצו **פרסם**

#### שינוי סדר פריטים

- ניתן לשנות את הסדר דרך **שאר ההגדרות → סדר** (Order) בעמוד עריכת הפריט
- מספר נמוך יותר = מופיע ראשון

#### הסרת פריט

- שלחו את הפריט לסל (Trash) או שנו את הסטטוס ל-"טיוטה"
- פריט שאינו פורסם לא יופיע בדף

#### קטגוריות תפריט

הקטגוריות מוגדרות מראש:
- פיצות | סלטים | מאפים | פסטות | שתייה | תוספות

לניהול קטגוריות: **SUN PEPE → פריטי תפריט → קטגוריות תפריט**

> **כשלא הוספתם עדיין פריטים** — הדף מציג תפריט דוגמה (Fallback) עם מחיר ₪0.
> ברגע שתוסיפו ותפרסמו פריט אחד לפחות, הדף יציג את הפריטים האמיתיים.

## שאלות נפוצות

**ש: המחירים לא מופיעים / כולם ₪0**
ת: ודאו ש-ACF מותקן ופעיל. ללא ACF המחיר תמיד יהיה ₪0.

**ש: שיניתי בהגדרות אבל לא רואה שינוי באתר**
ת: נסו לרענן את הדף של האתר (Ctrl+F5 או Cmd+Shift+R).

**ש: הדף לא נראה נכון / העיצוב שבור**
ת: ודאו שבחרתם בתבנית **SUN PEPE Landing Page** בעמוד הספציפי.

- כל ה-JS מוגבל לדף הנחיתה בלבד
